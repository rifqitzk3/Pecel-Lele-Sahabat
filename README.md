Pecel Lele Sahabat 
Website statis untuk Pecel Lele Sahabat, warung makan yang berlokasi di Jl. Malabar Ujung, Kota Bogor, Jawa Barat.
Fitur

Profil warung
Menu & harga
Galeri foto
Testimoni pelanggan
Pemesanan via WhatsApp (form popup modal)
Lokasi & Google Maps

Teknologi
HTML, CSS, JavaScript (vanilla)
Tanpa framework, tanpa backend
Responsif untuk mobile dan desktop

Struktur Folder
perweb/
├── index.html
├── styles.css
├── script.js
└── assets/
    ├── logo/
    ├── photos/
    └── gallery/
    
Cara Menjalankan
Cukup buka index.html di browser. Tidak perlu instalasi apapun.

Deploy
Website ini di-deploy menggunakan Netlify.

Dibuat sebagai proyek mata kuliah Pemrograman Web.
