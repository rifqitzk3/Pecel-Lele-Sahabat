## Deskripsi Proyek

Website ini dibuat untuk membantu **UMKM Pecel Lele Sahabat** dalam mempromosikan warung makan mereka secara online agar lebih mudah dikenal masyarakat luas. Website menyediakan informasi lengkap mengenai menu, harga, lokasi, jam buka, dan kontak pemesanan.

| Informasi | Detail |
|---|---|
| 📍 Lokasi | CR55+688, Jl. Malabar Ujung, RT.01/RW.07, Tegallega, Kec. Bogor Tengah, Kota Bogor |
| 🕙 Jam Buka | 10.00 – 22.00 setiap hari |
| 📱 Pemesanan | Via WhatsApp |

---

## Fitur Utama

| No | Fitur | Keterangan |
|---|---|---|
| 1 | **Profil Warung** | Informasi singkat tentang warung beserta foto |
| 2 | **Menu & Harga** | Daftar lengkap menu makanan dan minuman beserta harga |
| 3 | **Galeri Foto** | Foto warung dan sajian menu dalam layout grid |
| 4 | **Testimoni Pelanggan** | Ulasan dari pelanggan setia warung |
| 5 | **Form Pemesanan** | Popup modal untuk memilih menu dan jumlah porsi |
| 6 | **Integrasi WhatsApp** | Pesan otomatis terisi dan langsung dikirim ke WhatsApp warung |
| 7 | **Lokasi & Maps** | Tombol langsung menuju Google Maps lokasi warung |

---

## Teknologi yang Digunakan

- **HTML5** - Struktur halaman
- **CSS3** - Tampilan dan animasi (tanpa framework)
- **JavaScript (Vanilla)** - Interaksi form, modal, dan validasi
- Tidak menggunakan backend atau database - murni website statis

## Cara Menjalankan

Tidak memerlukan instalasi apapun. Cukup buka file `index.html` langsung di browser.

```bash
# Clone repository
git clone https://github.com/username/pecel-lele-sahabat.git

# Masuk ke folder
cd pecel-lele-sahabat

# Buka di browser
# Klik dua kali index.html, atau jalankan live server jika menggunakan VS Code
```

## Deploy

Website di-deploy secara gratis menggunakan **Netlify**.

🌐 **Live Demo:** [pecel-lele-sahabat.netlify.app](https://pecel-lele-sahabat.netlify.app)

Cara deploy ulang:
1. Buka [netlify.com](https://netlify.com)
2. Drag & drop folder `perweb` ke dashboard Netlify
3. Website langsung online
